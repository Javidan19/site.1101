This project was created as part of the SITE1101 course. The goal of the project is to understand the basic components of computer hardware and how they work together to process information.

Central Processing Unit (CPU)

The CPU is the main component of a computer responsible for processing instructions and performing calculations. It controls most operations inside the system.

Random Access Memory (RAM)

RAM is temporary memory that stores data while programs are running. It allows the computer to access information quickly and improves performance.

Storage

Storage devices such as hard drives and solid-state drives are used to store data permanently. The data remains saved even when the computer is turned off.
